# Network chatting in python

a Real time chatting system built with socket module in python.server and client can send eachother messages through network back and forth. 

## features 

- 2 way back and forth communcation on port 5555 locally with server to client model structure

- Real time communcation

- Timestamps on messages

## how to run

run server.py first and then client.py and it will automatically connect
